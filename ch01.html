1.4　リスト

連番つきリスト

連番つきでリストを使いたいときには`//list{...//}`を使います。

リスト1.2: 連番つきリストのサンプル

## 適当なサンプル
class Foo
  def foo!
    "foo!!!"
  end
end

リスト1.2: 連番つきリストのサンプル

## 適当なサンプル
class Foo
  def foo!
    "foo!!!"
  end
end

埋め込みリスト

連番なしで単純にリストを使いたいときには埋め込みリスト`//emlist{...//}`を使います。

// 適当なサンプル
class Foo {
  foo1() {
    return "foo!!!"
  }
}

見出しつきサンプル

// 適当なサンプル
class Foo {
  foo1() {
    return "foo!!!"
  }
}

1.5　embed

文字列をそのまま埋め込むには`//embed{...//}`を使います。

このままこのまま HTML・EPUB用です。HTMLタグも使えます。
